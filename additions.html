YoutubeId and YoutubeUrl - api

We took the youtubeId and the youtubeUrl(img of the song) by using youtube data api v3 call. we search in yourube query and get the result from there

we use this path : https://youtube.googleapis.com/youtube/v3/search?part=snippet&q=${encodeURIComponent(SongData.artistName + " " + SongData.name)}&key=${apiKey}.

the key we got from this web : https://console.cloud.google.com/apis/credentials?cloudshell=true&orgonly=true&project=tru-392715&supportedpurview=project.

we got the token access from this link : https://open.spotify.com/get_access_token?reason=transport&productType=web_player&_authfailed=1.
